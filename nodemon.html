nodemon

Monitor for any changes in your node.js application

nodemon is a utility that will monitor for any changes in your source and automatically restart your server. Perfect for development.

Get nodemon on GitHub

Features

  • Automatic restart on file changes - save time and improve your development workflow
  • Supports various extensions and types out of the box
  • Simple, zero-configuration notifier that just works
  • Cross-platform support to work seamlessly on Windows, macOS, and Linux
  • Configurable and extensible with advanced options for power users
